# AI Prompt — Build a Realistic Global Economic & Stock Market Simulator

Create a highly polished, realistic, large-scale web application called **“World Finance Simulator”** — a living simulation of the global economy, stock markets, geopolitics, corporations, supply chains, commodities, and financial systems.

The simulator should feel like a mix of:

* Capitalism Lab
* Victoria 3
* Cities: Skylines
* Wall Street Raider
* Bloomberg Terminal
* TradingView
* BlackRock level finance systems
* Tesla / Apple style company growth simulation

The world must feel alive, dynamic, reactive, and economically interconnected.

---

# CORE IDEA

Users enter a simulated Earth economy where:

* Every company has valuation, shares, employees, factories, products, supply chains, AI systems, growth metrics, risks, and competitors.
* Users can:

  * create companies
  * launch products
  * trade stocks
  * manipulate markets
  * acquire competitors
  * raise venture capital
  * build factories
  * manage logistics
  * influence governments
  * create monopolies
  * trigger economic wars

The simulation must continue evolving in real time even when users are offline.
